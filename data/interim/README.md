# data/interim

Intermediate files produced during coding. These files are working documents
and are not version-controlled (excluded via .gitignore).

## Files

- `extracted_case_notes.csv` — Raw extraction notes from primary source reading

## Purpose

These files capture the coder's working notes before cases are finalised in
`data/processed/csa_lite_cases.csv`. They support intercoder reliability
checks and audit trails but are not distributed as part of the public dataset.

## Access

Interim files are available upon reasonable request to the corresponding author
for replication purposes, subject to the absence of any confidential content.
