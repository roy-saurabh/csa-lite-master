# data/raw

Raw input files — never modified after initial collection.

## Files

- `candidate_cases.csv` — List of candidate cases identified during screening
- `sources_manifest.csv` — Full bibliography of all sources used

## Adding Cases

1. Add candidate cases to `candidate_cases.csv` following the field definitions
2. Add all sources to `sources_manifest.csv` with full bibliographic detail
3. Move accepted cases to `data/processed/csa_lite_cases.csv` after coding
4. Verify sources comply with the source quality rubric in `docs/source_quality_rubric.md`

## Data Ethics Note

All sources must be publicly accessible documents. No personal data should be
recorded in any field. See `docs/coding_protocol.md` for full guidance.
