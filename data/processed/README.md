# data/processed

Final processed datasets used for all analyses in the manuscript.

## Files

| File | Description |
|------|-------------|
| `csa_lite_cases.csv` | Coded case database (add real cases here manually) |
| `csa_lite_cases.jsonl` | Same data in JSONL format |
| `scored_cases.csv` | Generated by `csalite score`; includes all computed fields |
| `scored_cases.jsonl` | Same scored data in JSONL format |

## Workflow

```bash
# 1. After populating csa_lite_cases.csv with real coded cases:
csalite validate --input data/processed/csa_lite_cases.csv

# 2. Score the cases:
csalite score \
  --input data/processed/csa_lite_cases.csv \
  --output data/processed/scored_cases.csv \
  --jsonl-output data/processed/scored_cases.jsonl

# 3. Run full pipeline:
csalite all \
  --input data/processed/csa_lite_cases.csv \
  --outdir outputs/
```
