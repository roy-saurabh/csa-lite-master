# Data Directory

This directory contains all data for the CSA-lite project, organised into three stages.

## Structure

```
data/
  raw/          -- Original source files; never modified after collection
  interim/      -- Intermediate extraction notes (not version-controlled)
  processed/    -- Final processed datasets for analysis
```

## Data Ethics

CSA-lite uses only publicly documented information. No personal data is
collected, stored, or processed. All case records refer to AI system
deployments as reported in public records — not to individuals.

## Disclaimer

CSA-lite does not produce legal classifications, compliance determinations,
conformity assessments, or validated harm predictions. EU AI Act Annex III
categories are used only as an analytical reference frame for grouping
documented deployments by use area. The Context Severity Index is a
transparent structured-coding output derived from public-record evidence.
It is not a legal risk class.
