# outputs/reports/

Markdown reports generated by the CSA-lite pipeline.

| File | Description |
|------|-------------|
| `validation_report.md` | Validation results for the case dataset |
| `scoring_summary.md` | Scoring summary including band distributions and flags |

Generated by:
```bash
csalite validate --input data/processed/csa_lite_cases.csv --report outputs/reports/validation_report.md
csalite report --input data/processed/scored_cases.csv --out outputs/reports/scoring_summary.md
```
